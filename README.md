# Stockly
Stockly is your modern online store for essentials, trends, and unique finds—all always in stock. Enjoy fast shipping, secure checkout, and a seamless shopping experience. Stay stocked up with what you love, whenever you need it.

## Features

* Responsive UI with Tailwind CSS
* Typed React components using TypeScript
* MongoDB for data storage
* RESTful API integration

## Tech Stack

* Frontend: React, TypeScript, Tailwind CSS
* Backend: Node.js, Express (optional)
* Database: MongoDB


## Installation

Clone the repository:

```
git clone https://github.com/HenokCodez/Stockly.git
cd Stockly
```

Install dependencies:

```
cd client
npm install

cd ../server
npm install
```


## Usage

Start the client:

```
cd client
npm run dev
```

Start the server:

```
cd server
npm start
```


## Environment Variables

In the server folder, create a `.env` file:

```
MONGO_URI=your-mongodb-uri
JWT_SECRET=your-secret
```
